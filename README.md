# Martin's Moving

Static marketing site for Martin's Moving, a precision relocation company serving Bradenton, Sarasota, and Manatee County, FL.

## Stack
Plain static HTML with Tailwind via CDN. No build step. Deployed on Vercel.

## Pages
- `/` (index.html): home
- `/residential` (residential.html): residential moving
- `/commercial` (commercial.html): commercial moving
- `/contact` (contact.html): contact + request call back

`vercel.json` enables clean URLs, so `.html` extensions are dropped in production.
